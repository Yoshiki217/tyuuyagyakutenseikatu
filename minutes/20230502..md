## 企画書

```
企画 : ローカルブランドを伝えるwebサイト
ファッションプレスなどを筆頭に、様々なブランドのコレクションなどの写真が回っていますが、ローカルブランド(あまり有名ではない、有名になりたくない)などは、あまり取り上げてもらえず、商品を卸しているショップでしか目にすることができないものをwebサイトで見えるようにしたい。

概要
WEBサイトで、ユーザーはローカルブランドのコレクション LOOK(服の画像)などを、みることができる
逆に、ブランド側にはブランドの登録とLOOKのアップロード機能とブランドコンセプトを伝えてもらう

将来的
日本のマイナーブランドをアジアを中心に指名買いされ続ける未来を作る


技術
フロント : html . Css . JavaScript
バック : PHP or Node or Laravel
インフラ : AWS or Cloudflare or alibaba Cloud
```
